# 🚀 Nexus Business Platform

Full-featured AI-powered business management platform — built as a single HTML file with React.

## Features
- CRM, Inventory, Accounting, HR, Manufacturing
- Bilingual (Arabic / English)
- Dark / Light mode
- Auth (signup, login, forgot password)
- AI Chatbot (Claude API)
